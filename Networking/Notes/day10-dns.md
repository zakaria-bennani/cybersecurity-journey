# Day 10 DNS

## Concepts Learned

- DNS
- Domain Names
- A Records
- AAAA Records
- MX Records
- CNAME Records

## Commands Practiced

- nslookup
- dig
- dig +short

## DNS Resolution Process

1. User enters domain
2. DNS server finds IP address
3. IP address returned
4. Browser connects to server
5. Website loads

## Security Insight

DNS is frequently targeted in phishing, malware communication, and spoofing attacks.

## Reflection

Today I learned how domain names are translated into IP addresses and how DNS makes the Internet usable.
