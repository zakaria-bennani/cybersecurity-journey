# Network Information Tool

## Description

A Bash-based networking utility that gathers information about a Linux system.

## Features

- Displays hostname
- Displays IP addresses
- Displays default gateway
- Displays DNS configuration
- Displays listening ports
- Displays network interfaces

## Usage

```bash
chmod +x networkinfo.sh
./networkinfo.sh
```

## Technologies Used

- Linux
- Bash
- Networking Commands

## Purpose

Created as part of a cybersecurity networking fundamentals learning roadmap.
