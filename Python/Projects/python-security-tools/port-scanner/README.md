# Python Port Scanner

## Description

A Python TCP port scanner built using sockets.

## Features

- TCP scanning
- Port range selection
- CLI arguments
- Report generation
- Error handling

## Usage

```bash
python3 scanner.py \
--target 127.0.0.1 \
--start 1 \
--end 1024
```

## Security Relevance

Port scanning is used by security engineers to identify exposed services and assess attack surfaces.

Only scan systems you own or have permission to test.
