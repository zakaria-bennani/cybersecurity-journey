# Security Toolkit

A Python-based cybersecurity toolkit.

## Modules

### System Information

Collects:
- User
- Hostname
- OS Information
- Memory Usage

### Port Scanner

Performs TCP port scans.

### Log Analyzer

Analyzes authentication logs.

### Password Checker

Evaluates password strength.

## Usage

python toolkit.py sysinfo

python toolkit.py portscan

python toolkit.py analyze-log

python toolkit.py password-check

## Security Relevance

Demonstrates:

- Python
- Networking
- Automation
- Security Analysis
- CLI Tool Development
