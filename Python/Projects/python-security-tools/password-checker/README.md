# Password Strength Checker

## Description

A Python security tool that evaluates password strength.

## Features

- Length analysis
- Uppercase detection
- Lowercase detection
- Number detection
- Special character detection
- Common password detection
- Report generation

## Usage

python3 password_checker.py

or

python3 password_checker.py --password Password123!

## Security Relevance

Weak passwords remain one of the most common attack vectors in cybersecurity.
