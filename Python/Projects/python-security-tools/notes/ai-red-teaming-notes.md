# AI Red Teaming Notes

AI Red Teaming is the practive of testing AI systems for vulnrabilities, unsafe behavior, abuse cases, and security weaknesses.

Goal:

Find problems before attackers do. 

## Why It Interests Me?
 
I enjoy offensive security and AI.

AI Red Teaming combines both of these fields and appears likely to grow significantly over the next decade.
