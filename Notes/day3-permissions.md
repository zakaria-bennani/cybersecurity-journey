# Day 3 Linux Permissions

## Concepts Learned

- Users
- Groups
- Others
- Read
- Write
- Execute

## Commands Practiced

- whoami
- id
- ls -l
- chmod
- chown

## Numeric Permissions

644 = rw-r--r--

755 = rwxr-xr-x

600 = rw-------

777 = rwxrwxrwx

## Security Insight

Permissions control who can access files. Poor permissions can create security vulnerabilities.
