# Prompt Injection Tester

## Overview

This project evaluates how a Large Language Model responds to several categories of prompt manipulation.

## Purpose

The goal is educational AI security research.

It is designed to observe model behavior rather than bypass safety mechanisms.

## Attack Categories

- Instruction Override
- Role Manipulation
- Context Manipulation
- Delimiter Attacks

## Features

- Automated testing
- Structured logging
- Timestamped reports
- Modular attack library

## Technologies

- Python
- Gemini API
- argparse
- File handling

## Ethical Notice

This project is intended for authorized security testing, education, and research only.
